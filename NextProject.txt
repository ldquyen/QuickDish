Dự án tiếp theo về phần order món ăn, demo cho việc mở quán ăn quy mô nhỏ

2 bảng chính lưu trên MockAPI và 1 bảng dùng để lưu trong ORDER:
- Menu:
    MenuID: string     
    Name: string
    Description: string
    Category: string                //Món chính | Món phụ | Đồ uống
    Price: number
    Quantity: number 
    URLImage: string
    Ingredients: string             //nguyên liệu chi tiết
    IsActive: boolean
- Order:
    OrderID: string
    Table: string
    Items: array<ItemDetail>[]
    TotalAmount: number
    Status: string                  //Đang xử lý | Đang phục vụ | Đã thanh toán 
    CreatedAt: date
    UpdateAt: date
    Note: string
- ItemDetail:
    MenuID: string
    Name: string
    Quantity: number
    Price: number
    TotalPrice: number

- Trang MENU:
    + trang MENU có api để lấy lên danh sách món ăn, truyền vào MenuCard để hiển thị 
    + trong MenuCard có nút ORDER 
    => đặt món sao
- Trang CHECKOUT:
    + trang CHECKOUT có nút EDIT của 1 component modal, khi bấm thì truyền data từ CHECKOUT vào để hiển thị,
    nhưng sau khi update thành công thì làm sao update lại bên trang checkout là cái mới nhất
